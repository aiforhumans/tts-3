# Advanced Local TTS Studio Setup

## 1️⃣ Install dependencies:

pip install -r requirements.txt

## 2️⃣ Run the app:

python app_v3.py

## 3️⃣ Add cloned voices:

Put WAV files in /clones/ folder.
They will appear as "Clone: Name" in speaker dropdown.

## Notes:

- Use Gradio 4.38.0 (included in requirements.txt).
- Works with Coqui TTS + OpenVoice wrapper.
